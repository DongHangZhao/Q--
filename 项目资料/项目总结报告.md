# 咫尺天涯社交平台 - 项目总结报告

## 项目概述
咫尺天涯社交平台是一个基于Flask框架的完整社交网络应用，集成了用户管理、内容分享、即时通讯、新闻资讯、热度排行等多项功能。

## 项目架构
- **前端**: HTML/CSS/JavaScript + Bootstrap 5 响应式设计
- **后端**: Python Flask Web框架
- **数据库**: SQLite
- **功能模块**: 用户系统、帖子系统、视频系统、消息系统、新闻系统、热度排行系统

## 核心功能
1. **用户系统**: 注册、登录、个人资料管理、头像上传
2. **内容系统**: 发布帖子、视频上传与播放
3. **社交功能**: 用户关注/粉丝、点赞、评论
4. **消息系统**: 私信聊天、附件传输
5. **新闻系统**: 新闻浏览、点赞、评论
6. **热度排行**: 基于算法的每日、每周、每月热度排行
7. **管理员系统**: 用户管理、内容审核、封禁功能

## 技术特点
- 采用MVC架构模式
- 实现了完整的用户认证和授权机制
- 支持多媒体内容上传和处理
- 实现实时消息推送功能
- 采用响应式设计适配多设备

## 数据库设计
- users: 用户信息表
- posts: 帖子信息表
- videos: 视频信息表
- messages: 消息表
- news: 新闻信息表
- follows: 关注关系表
- likes: 点赞表
- comments: 评论表
- trending: 热度排行表

## 部署说明
1. 安装Python 3.x
2. 安装依赖: pip install -r requirements.txt
3. 初始化数据库: python init_db.py
4. 启动应用: python run.py
5. 访问地址: http://localhost:5000
