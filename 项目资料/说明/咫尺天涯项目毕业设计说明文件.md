# 咫尺天涯社交平台 - 毕业设计说明文件

## 项目概述

咫尺天涯社交平台是一个基于Python Flask框架开发的综合性社交应用，旨在为用户提供集新闻、视频、帖子分享、聊天于一体的综合社交体验。平台名称寓意着虽然人与人之间可能相隔万里，但通过网络可以实现心灵的贴近交流。

### 项目目标
- 实现完整的用户注册、登录、个人信息管理功能
- 提供内容发布与分享平台（帖子、视频、新闻）
- 构建用户间的社交网络（关注、点赞、评论、私信）
- 实现智能热度排行算法
- 提供响应式的用户界面，适配桌面端和移动端

### 技术栈
- **后端**: Python Flask 2.x
- **数据库**: SQLite3 + SQLAlchemy ORM
- **前端**: HTML5, CSS3, JavaScript, Bootstrap 5
- **模板引擎**: Jinja2
- **表单处理**: Flask-WTF
- **用户认证**: Flask-Login

## 核心功能模块

### 1. 用户系统
- **用户注册/登录**: 基于Flask-Login的安全认证系统
- **个人信息管理**: 用户可以管理头像、简介、位置等信息
- **权限控制**: 支持普通用户和管理员角色

### 2. 内容系统
- **帖子功能**: 用户可以发布图文帖子，支持点赞和评论
- **视频功能**: 支持视频上传、播放和点赞
- **新闻功能**: 自动获取和展示时事新闻

### 3. 社交互动
- **关注系统**: 用户间可以相互关注
- **点赞系统**: 对帖子、视频、新闻进行点赞
- **评论系统**: 发布和管理评论
- **私信聊天**: 用户间一对一私信功能

### 4. 管理功能
- **管理员系统**: 支持管理员账户管理
- **用户权限控制**: 可以封禁用户或限制其特定功能
- **内容审核**: 管理员可以管理平台内容

### 5. 智能功能
- **热度排行**: 基于点赞数、评论数和发布时间的智能热度算法
- **内容历史**: 记录内容的修改历史
- **用户状态**: 实时显示用户在线状态

## 数据库设计

### 主要数据模型
- **User**: 用户模型，包含基本信息、权限设置等
- **Post**: 帖子模型，支持图文内容
- **Video**: 视频模型，支持视频上传和管理
- **News**: 新闻模型，自动获取新闻内容
- **Comment**: 评论模型，支持多类型内容评论
- **Message**: 私信模型，支持用户间消息传递
- **Follows**: 关注关系模型
- **Likes**: 点赞模型（帖子、视频、新闻）
- **Trending**: 热度排行模型
- **ContentHistory**: 内容修改历史模型

### 数据库优化特性
- 使用自定义FlexibleDateTime类型处理多种时间戳格式
- 建立适当的索引提高查询效率
- 合理的关系映射减少冗余数据

## 技术实现亮点

### 1. 时间戳兼容性处理
由于数据库中存在多种时间戳格式，实现了FlexibleDateTime类型装饰器，能够处理字符串、数字和标准datetime格式的时间戳。

### 2. 智能热度算法
实现了综合考虑点赞数、评论数、浏览量和内容年龄的热度评分算法，公式如下：
```
基础分数 = (点赞数 × 3) + (浏览数 × 0.5) + (评论数 × 2)
衰减因子 = max(0.1, 24.0 / (内容年龄小时数 + 1))
最终得分 = 基础分数 × 衰减因子
```

### 3. 响应式界面设计
使用Bootstrap 5构建响应式界面，确保在不同设备上都有良好的用户体验。

### 4. 安全措施
- 密码使用哈希加密存储
- CSRF保护
- 输入验证和过滤
- 权限访问控制

## 部署与运行

### 环境要求
- Python 3.7+
- pip包管理器

### 安装步骤
1. 克隆或下载项目到本地
2. 创建虚拟环境（推荐）：
   ```
   python -m venv .venv
   .venv\Scripts\activate  # Windows
   source .venv/bin/activate  # Linux/Mac
   ```
3. 安装依赖：
   ```
   pip install -r requirements.txt
   ```
4. 启动应用：
   ```
   python run.py
   ```
5. 访问 `http://localhost:5000`
