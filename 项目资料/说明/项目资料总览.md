# 咫尺天涯项目资料总览

## 项目概览
咫尺天涯社交平台是一个基于Python Flask框架开发的综合性社交应用，包含用户系统、内容发布、社交互动、管理员功能等完整功能。

## 项目文件结构

### 核心项目文件 (位于 python_web_project/)
- **app.py** - Flask主应用文件
- **config.py** - 配置管理文件  
- **run.py** - 应用启动脚本
- **utils.py** - 工具函数模块
- **requirements.txt** - 项目依赖声明
- **README.md** - 项目说明文档
- **app.db** - 主数据库文件

## 项目特色功能

### 1. 智能热度算法
基于点赞数、评论数、浏览量和内容发布时间的综合热度评分算法。

### 2. 用户状态管理
实时用户在线状态检测和好友状态同步功能。

### 3. 内容历史追踪
记录内容修改历史，支持版本对比和权限控制。

### 4. 多媒体支持
支持图片、视频等多种媒体格式的内容发布和管理。

### 5. 完善的权限系统
管理员可以精细化控制用户权限，包括封禁、限制特定功能等。

## 技术栈

### 后端技术
- **框架**: Flask 2.x
- **ORM**: SQLAlchemy
- **数据库**: SQLite3
- **认证**: Flask-Login
- **表单**: Flask-WTF

### 前端技术
- **框架**: Bootstrap 5
- **语言**: HTML5, CSS3, JavaScript
- **模板**: Jinja2

### 部署环境
- **Python版本**: 3.7+
- **依赖管理**: pip + requirements.txt
- **部署方式**: 直接运行Python脚本

## 项目运行

### 环境要求
- Python 3.7+
- pip包管理器

### 安装运行
1. 安装依赖: `pip install -r requirements.txt`
2. 启动应用: `python run.py`
3. 访问: `http://localhost:5000`

## 项目意义

咫尺天涯社交平台项目展示了现代Web开发的最佳实践，包括完整的用户系统、内容管理、社交功能、权限控制等，为用户提供了一个功能丰富、安全可靠的社交环境。项目不仅实现了预定的功能目标，还在性能优化、用户体验、安全防护等方面进行了深入的思考和实现。
