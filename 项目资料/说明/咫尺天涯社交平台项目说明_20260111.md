# 咫尺天涯社交平台项目说明

## 项目概述
咫尺天涯社交平台是一个基于Python Flask框架开发的全功能社交网络应用，旨在为用户提供内容分享、视频观看、用户互动等多种社交功能。

## 技术架构
- **后端框架**: Flask (Python)
- **数据库**: SQLite
- **前端框架**: Bootstrap 5 (响应式设计)
- **编程语言**: Python
- **模板引擎**: Jinja2

## 核心功能模块

### 1. 用户系统
- 用户注册与登录
- 个人资料管理
- 头像上传与管理
- 用户间关注/粉丝关系

### 2. 内容发布系统
- 文章发布与编辑
- 视频上传与管理
- 新闻资讯发布
- 配图上传功能

### 3. 互动功能
- 评论系统
- 点赞功能
- 私信功能
- 用户关注/取消关注

### 4. 视频系统
- 视频上传与播放
- 自动封面生成
- 视频缩略图管理
- 视频评论功能

### 5. 热度排行系统
- 每日、每周、每月热度排行
- 基于点赞数、评论数、观看数的算法
- 时间衰减因子计算

### 6. 管理功能
- 用户权限管理
- 内容审核
- 封禁功能

## 项目特色功能
- 自动视频封面生成：上传视频时若未提供缩略图，系统自动从视频中提取帧作为封面
- 响应式设计：适配移动端和桌面端设备
- 本土化适配：支持中文界面和国内用户习惯
- 浏览器兼容：使用mp4v编码确保视频在各浏览器正常播放

## 目录结构
```
python_web_project/
├── app.py                 # 主应用文件
├── config.py             # 配置文件
├── models/               # 数据模型
├── forms/                # 表单定义
├── utils/                # 工具函数
├── static/               # 静态资源(css, js, images)
├── templates/            # HTML模板
├── requirements.txt      # 依赖包列表
└── run.py                # 启动脚本
```

## 安装与部署
1. 安装依赖：`pip install -r requirements.txt`
2. 初始化数据库：`python initialize_data.py`
3. 启动应用：`python run.py`
4. 访问：http://localhost:5000

## 开发规范
- 所有媒体文件本地化存储
- 使用浏览器友好的视频编码格式
- 实现响应式设计适配移动端
- 统一管理动态资源URL
