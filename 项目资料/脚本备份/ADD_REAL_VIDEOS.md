# 添加真实国内短视频指南

本项目现已支持添加来自国内视频网站的真实短视频。以下是详细的操作指南：

## 方法一：使用第三方工具下载视频

### 1. 使用视频下载工具

推荐使用以下工具从国内视频网站下载视频：

#### 抖音视频下载
- 使用抖音网页版复制视频链接
- 使用在线解析工具（如 douyin.wtf）解析下载
- 或使用专门的抖音下载软件

#### B站视频下载
- 使用 Bilibili-Evolved 浏览器插件
- 或使用 youtube-dl / yt-dlp 工具
- 命令示例：`yt-dlp [视频URL]`

#### 其他平台
- 西瓜视频、快手等平台也有相应的下载工具

### 2. 准备视频文件

- 视频格式：MP4、AVI、MOV 等常见格式
- 视频大小：建议小于 100MB
- 视频时长：建议 10-60 秒（适合短视频）

## 方法二：运行处理脚本

### 1. 创建下载目录
```bash
mkdir downloads
```

### 2. 将下载的视频放入 downloads 目录
```
项目根目录/
├── downloads/
│   ├── video1.mp4
│   ├── video2.mp4
│   └── video3.mp4
```

### 3. 运行处理脚本
```bash
python handle_manual_videos.py
```

脚本会自动：
- 识别 downloads 目录中的视频文件
- 为每个内置用户分配一个视频
- 生成对应的缩略图
- 更新数据库记录

## 视频格式要求

- **格式**：MP4（推荐）、AVI、MOV、WMV、MKV
- **分辨率**：建议 640x480 或 720x480
- **帧率**：24fps 或 30fps
- **编码**：H.264 或其他浏览器兼容编码
- **大小**：建议小于 100MB

## 验证视频播放

运行项目后，在浏览器中访问视频页面，检查视频是否能正常播放：

```bash
python app.py
```

访问 `http://localhost:5000/videos` 查看视频列表

## 故障排除

### 视频无法播放
1. 检查视频编码格式是否被浏览器支持
2. 确认视频文件完整无损坏
3. 检查文件路径是否正确

### 缩略图不显示
1. 确认 JPG 文件存在
2. 检查文件权限

## 自动处理脚本功能

如果未在 downloads 目录找到视频文件，脚本会自动：
- 创建高质量的本地视频（使用 OpenCV 生成）
- 确保视频可在浏览器中播放
- 保持所有功能完整

## 注意事项

1. 请遵守各视频平台的版权规定
2. 仅用于学习和演示目的
3. 确保网络连接稳定以下载视频
4. 视频文件需要符合网站上传规范
