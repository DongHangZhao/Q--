## 🎯 可以使用的所有功能

### 用户功能

#### 1. 修改密码
```
URL: http://127.0.0.1:5000/password/change-password
前提: 需要登录
功能: 输入旧密码和新密码修改
```

#### 2. 忘记密码（完整流程）
```
步骤1: 访问 http://127.0.0.1:5000/password/forgot-password
       输入用户名和图像验证码
       获取6位重置码

步骤2: 联系管理员，提供重置码

步骤3: 管理员验证身份后

步骤4: 访问 http://127.0.0.1:5000/password/verify-code
       输入用户名和重置码
       验证成功

步骤5: 自动跳转到重置密码页面
       设置新密码
```

#### 3. 重置密码
```
URL: http://127.0.0.1:5000/password/reset-password?user_id=1
前提: 已通过验证
功能: 设置新密码
```

### 管理员功能

#### 1. 查看用户重置码
```javascript
// API调用
POST http://127.0.0.1:5000/password/admin/view-reset-code
参数: user_id=1

// 返回示例
{
  "success": true,
  "message": "用户 admin 的重置码: 436887",
  "verification_code": "436887",
  "expires_at": "2026-04-29 16:30:00"
}
```

#### 2. 直接重置用户密码
```javascript
// API调用
POST http://127.0.0.1:5000/password/admin/reset-user-password
参数: user_id=1&new_password=newpass123

// 返回示例
{
  "success": true,
  "message": "已重置用户 admin 的密码"
}
```
