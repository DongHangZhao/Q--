#### 2. 重置密码功能优化 ✅
**问题**：重置密码后密码显示不够清晰
**优化**：
- 生成更安全的密码（字母+数字组合，8位）
- 在flash消息中明确显示新密码
- 提示"此密码只显示一次"
- 使用session临时存储密码（备用）

**密码生成规则**：
```python
chars = 'abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789'
new_password = ''.join([random.choice(chars) for _ in range(8)])
# 示例：aB3xK9mP, 7qR2nT8w
```

**验证**：
1. 访问 http://127.0.0.1:5000/admin/users
2. 点击用户的"重置密码"按钮
3. 页面顶部显示：✅ 已重置用户 xxx 的密码为: aB3xK9mP，请立即通知用户！此密码只显示一次

- 位置：操作菜单

  - **禁止发帖** - 只能发视频和评论

**智能显示逻辑**：
```jinja2

  显示"解除所有封禁"按钮
{% else %}
  显示"限制功能"下拉菜单
{% endif %}
```
