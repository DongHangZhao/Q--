#### 2. ✅ 注册时间显示问题
**问题**：数据库后台（dbadmin）注册时间列显示为空

**原因**：column_list使用的是`created_at`，但User模型字段是`join_date`

**修复**：
```python
# 修改前
column_list = (..., 'created_at')
column_labels = {'created_at': '注册时间'}

# 修改后
column_list = (..., 'join_date')
column_labels = {'join_date': '注册时间'}
```

**验证**：访问 http://127.0.0.1:5000/dbadmin/user/，注册时间正常显示

---

#### 3. ✅ 下拉菜单跳转页面顶部
**问题**：点击"限制功能"下拉菜单会跳转到页面顶部

**原因**：使用`href="#"`会触发页面跳转

**修复**：
```html
<!-- 修改前 -->
<a href="#" class="dropdown-toggle" data-toggle="dropdown">

<!-- 修改后 -->
<a href="javascript:void(0);" class="dropdown-toggle" data-bs-toggle="dropdown">
```

**验证**：点击下拉菜单不再跳转页面

## 🔧 技术细节

### Bootstrap 5下拉菜单语法
```html
<li class="dropdown">
    <a href="javascript:void(0);" 
       class="dropdown-toggle" 
       data-bs-toggle="dropdown" 
       aria-expanded="false">
        限制功能 <span class="caret"></span>
    </a>
    <ul class="dropdown-menu" style="min-width: 150px;">
        <li>
            <a href="..." class="dropdown-item">
                选项1
            </a>
        </li>
        <li><hr class="dropdown-divider"></li>
        <li>
            <a href="..." class="dropdown-item">
                选项2
            </a>
        </li>
    </ul>
</li>
```

### 关键点
1. `href="javascript:void(0);"` - 防止页面跳转
2. `data-bs-toggle="dropdown"` - Bootstrap 5语法
3. `dropdown-item` - 菜单项样式
4. `dropdown-divider` - 分隔线
5. `min-width` - 最小宽度控制
