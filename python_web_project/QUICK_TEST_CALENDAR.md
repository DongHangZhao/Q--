## 💡 关键修复点

### 问题根源
```python
# calendar.monthcalendar() 返回的周列表中，空白天数用 0 表示
[0, 0, 0, 0, 0, 1, 2]  # 前 5 天是上月的空白
```

### 之前的错误代码
```jinja2
calendar_module.weekday(year, month-1, day-1)  
# 当 day=0 时，变成 weekday(..., -1)，报错！
```

### 现在的正确代码
```jinja2
{% set is_weekend = false %}
{% if day != 0 %}
  {% set is_weekend = calendar_module.weekday(year, month-1, day-1) >= 5 %}
{% endif %}
# 先检查 day!=0，再调用 weekday()，安全！
```
