### 3. 核心功能实现 ✅

#### 用户修改密码
```python
POST /password/change-password
参数：
  - old_password: 旧密码
  - new_password: 新密码
  - confirm_password: 确认密码
  
功能：
  1. 验证旧密码
  2. 验证新密码长度（>=6位）
  3. 验证两次密码一致
  4. 更新密码
  5. 记录操作日志
```

#### 忘记密码流程
```python
步骤1: POST /password/forgot-password
  参数：username_or_email
  功能：
    1. 查找用户
    2. 生成6位验证码
    3. 创建重置请求（30分钟有效）
    4. 通知管理员（需集成邮件/短信）

步骤2: POST /password/verify-code
  参数：username, code
  功能：
    1. 验证用户名和验证码
    2. 检查验证码是否有效
    3. 标记验证码为已使用
    4. 返回user_id用于重置

步骤3: POST /password/reset-password
  参数：user_id, new_password, confirm_password
  功能：
    1. 验证新密码
    2. 更新密码
    3. 记录操作日志
```

#### 管理员功能
```python
管理员重置密码: POST /password/admin/reset-user-password
  参数：user_id, new_password
  功能：
    1. 验证管理员权限
    2. 重置用户密码
    3. 记录操作日志

获取验证码: POST /password/admin/send-verification-code
  参数：user_id
  功能：
    1. 验证管理员权限
    2. 查找用户的重置请求
    3. 返回验证码（实际项目应发送邮件/短信）
```

## 📋 使用流程

### 场景1：用户修改密码

**步骤**：
1. 用户登录系统
2. 访问 `http://127.0.0.1:5000/password/change-password`
3. 输入：
   - 旧密码
   - 新密码
   - 确认新密码
4. 点击"修改密码"
5. 修改成功，自动跳转首页

**API调用示例**：
```javascript
fetch('/password/change-password', {
  method: 'POST',
  headers: {'Content-Type': 'application/x-www-form-urlencoded'},
  body: 'old_password=123456&new_password=newpass&confirm_password=newpass'
})
.then(res => res.json())
.then(data => {
  if (data.success) {
    alert('密码修改成功');
  }
});
```

### 场景2：用户忘记密码

**步骤1：提交申请**
1. 访问 `http://127.0.0.1:5000/password/forgot-password`
2. 输入用户名或邮箱
3. 点击"提交重置请求"
4. 系统生成验证码并通知管理员

**步骤2：获取验证码**
1. 管理员收到通知
2. 访问管理员面板查看验证码
3. 通过邮件/短信发送给用户

**步骤3：验证验证码**
1. 访问 `http://127.0.0.1:5000/password/verify-code`
2. 输入用户名和收到的验证码
3. 点击"验证"

**步骤4：设置新密码**
1. 验证成功后跳转到重置密码页面
2. 输入新密码并确认
3. 点击"重置密码"
4. 密码重置成功

### 场景3：管理员重置用户密码

**步骤**：
1. 管理员登录
2. 在用户管理页面选择用户
3. 调用API：
```javascript
fetch('/password/admin/reset-user-password', {
  method: 'POST',
  body: 'user_id=1&new_password=temp123456'
})
```
4. 将临时密码发送给用户
5. 用户登录后修改密码

## 🔧 API接口文档

### 1. 用户修改密码
```
POST /password/change-password
需要登录: 是

请求体:
  old_password: string (必填)
  new_password: string (必填, >=6位)
  confirm_password: string (必填)

响应:
  成功: {success: true, message: "密码修改成功"}
  失败: {success: false, message: "错误原因"}
```

### 2. 忘记密码申请
```
POST /password/forgot-password
需要登录: 否

请求体:
  username_or_email: string (必填)

响应:
  成功: {success: true, message: "请求已提交"}
  失败: {success: false, message: "用户不存在"}
```

### 3. 验证验证码
```
POST /password/verify-code
需要登录: 否

请求体:
  username: string (必填)
  code: string (必填, 6位)

响应:
  成功: {success: true, user_id: 1}
  失败: {success: false, message: "验证码无效"}
```

### 4. 重置密码
```
POST /password/reset-password
需要登录: 否

请求体:
  user_id: string (必填)
  new_password: string (必填, >=6位)
  confirm_password: string (必填)

响应:
  成功: {success: true, message: "密码重置成功"}
  失败: {success: false, message: "错误原因"}
```

### 5. 管理员重置密码
```
POST /password/admin/reset-user-password
需要登录: 是
需要权限: 管理员

请求体:
  user_id: string (必填)
  new_password: string (必填, >=6位)

响应:
  成功: {success: true, message: "已重置用户密码"}
  失败: {success: false, message: "无权操作"}
```

### 6. 获取验证码
```
POST /password/admin/send-verification-code
需要登录: 是
需要权限: 管理员

请求体:
  user_id: string (必填)

响应:
  成功: {
    success: true, 
    verification_code: "123456",
    message: "验证码: 123456"
  }
  失败: {success: false, message: "错误原因"}
```
