## 🎯 使用流程示例

### 场景1：用户忘记密码

**步骤1**: 访问登录页面
```
http://127.0.0.1:5000/login
```
点击"忘记密码？"链接

**步骤2**: 进入忘记密码页面
```
http://127.0.0.1:5000/password/forgot-password
```
- 输入用户名
- 完成图像验证码
- 提交获取6位重置码

**步骤3**: 联系管理员
- 提供重置码
- 管理员验证身份

**步骤4**: 验证重置码
```
http://127.0.0.1:5000/password/verify-code
```
- 输入用户名
- 输入6位重置码
- 验证成功

**步骤5**: 设置新密码
```
http://127.0.0.1:5000/password/reset-password
```
- 输入新密码
- 确认新密码
- 完成重置

---

http://127.0.0.1:5000/admin/users

## 📊 数据库表

### password_reset_requests
```sql
- id: 主键
- user_id: 用户ID
- verification_code: 6位重置码
- created_at: 创建时间
- expires_at: 过期时间（30分钟）
- is_used: 是否已使用
- used_at: 使用时间
```

### password_change_logs
```sql
- id: 主键
- user_id: 用户ID
- changed_by: 修改者（user/admin/system）
- ip_address: IP地址
- timestamp: 时间
- reason: 修改原因
```

## 🔒 安全特性

1. **密码加密**: 使用Werkzeug加密存储
2. **图像验证码**: 防止机器人攻击
3. **6位重置码**: 30分钟有效，一次性使用
4. **操作日志**: 完整记录所有操作
5. **权限控制**: 管理员功能需要权限验证
6. **密码要求**: 最少6位
