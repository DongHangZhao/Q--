# 密码管理系统 - 完整实现说明

## ✅ 已实现功能

### 1. 数据库模型

已添加以下模型到 `models/__init__.py`:

#### PasswordResetRequest - 密码重置请求
```python
- user_id: 用户ID
- verification_code: 6位验证码
- created_at: 创建时间
- expires_at: 过期时间（30分钟）
- is_used: 是否已使用
- used_at: 使用时间
```

#### PasswordChangeLog - 密码修改日志
```python
- user_id: 用户ID
- changed_by: 修改者 ('user', 'admin', 'system')
- ip_address: IP地址
- timestamp: 修改时间
- reason: 修改原因
```

### 2. 路由功能

已创建 `routes/password_routes.py`，包含以下路由：

#### 用户功能
1. **`/password/change-password`** - 用户修改密码
   - 需要登录
   - 验证旧密码
   - 输入新密码并确认
   
2. **`/password/forgot-password`** - 忘记密码
   - 输入用户名或邮箱
   - 提交重置请求
   - 生成验证码
   
3. **`/password/verify-code`** - 验证验证码
   - 输入用户名和验证码
   - 验证是否正确
   
4. **`/password/reset-password`** - 重置密码
   - 输入新密码
   - 确认新密码

#### 管理员功能
1. **`/password/admin/reset-user-password`** - 管理员重置用户密码
   - 需要管理员权限
   - 直接设置新密码
   
2. **`/password/admin/send-verification-code`** - 发送验证码
   - 查看用户的验证码
   - 可以手动发送给用户

    </form>

<script>
document.getElementById('changePasswordForm').addEventListener('submit', async (e) => {
    e.preventDefault();
    const formData = new FormData(e.target);
    
    const response = await fetch('/password/change-password', {
        method: 'POST',
        body: formData
    });
    
    const result = await response.json();
    
    if (result.success) {
        alert('密码修改成功');
        window.location.href = '/';
    } else {
        alert(result.message);
    }
});
</script>
{% endblock %}

## 📝 使用说明

### 用户修改密码流程
1. 登录后访问 `/password/change-password`
2. 输入旧密码和新密码
3. 点击"修改密码"
4. 修改成功后自动跳转到首页

### 忘记密码流程
1. 访问 `/password/forgot-password`
2. 输入用户名或邮箱
3. 提交重置请求
4. 管理员收到请求并发送验证码
5. 用户访问 `/password/verify-code` 输入验证码
6. 验证成功后设置新密码

### 管理员重置密码流程
1. 管理员登录
2. 访问用户管理页面
3. 选择用户并点击"重置密码"
4. 输入新密码
5. 系统记录操作日志

## 🔒 安全特性

1. **密码加密**：使用Werkzeug的generate_password_hash
2. **验证码有效期**：30分钟后自动过期
3. **一次性验证码**：使用后标记为已使用
4. **操作日志**：所有密码修改都记录在案
5. **权限控制**：管理员功能需要is_admin权限
