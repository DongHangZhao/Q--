# 新闻采集系统使用说明

## ✅ 已实现功能

### 1. 每日自动采集（应用启动时执行）
- **澎湃新闻API**：获取20条热点新闻
- **多源爬虫**：百度、搜狗、搜狐、第一财经、财新等10+网站
- **自动去重**：基于标题去重
- **真实链接**：所有新闻带原文链接

### 2. 采集效果
- **今天（2026-04-11）**：780条真实新闻
- **来源**：澎湃新闻、搜狐、第一财经、财新网、钛媒体、南方日报等
- **带URL**：可跳转到原新闻页面

### 3. 脚本说明

#### daily_news_update.py - 每日更新
```bash
# 手动执行
python daily_news_update.py

# 应用启动时自动执行
python run.py
```

#### scrape_the_paper.py - 澎湃新闻专用
```bash
python scrape_the_paper.py
```

#### real_news_scraper.py - 多源爬虫
```bash
# 爬取今天
python real_news_scraper.py --today

# 自动补充缺失日期
python real_news_scraper.py --auto-fill
```

## ❌ 无法实现的功能

### 历史新闻（2026年1-3月）
**原因**：
1. 新闻网站不开放历史归档
2. 搜索引擎只索引最近30天
3. 历史新闻需要付费API

**替代方案**：
- 使用新闻API（约50元/月）
- 手动导入数据集（CSV/JSON）
- 只用最近30天的真实新闻

## ⚠️ 注意事项

1. **反爬机制**：每次采集间隔1-3秒
2. **去重机制**：基于标题前40字符
3. **链接有效性**：部分新闻链接可能失效
4. **采集频率**：建议每天1-2次

## 📈 优化建议
