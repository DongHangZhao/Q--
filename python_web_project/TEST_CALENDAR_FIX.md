## 🔍 常见问题排查

### Q1: 还是报错 ValueError
**可能原因**：
- 浏览器缓存了旧页面
- 应用没有重启

**解决方法**：
1. 强制刷新页面（Ctrl+F5）
2. 清除浏览器缓存
3. 重启应用

### Q2: 所有日期都不可点击
**可能原因**：
- 数据库中没有该日期的新闻
- 新闻数据时间戳不对

**解决方法**：
```bash
# 检查数据库中的新闻
sqlite3 users.db
SELECT DATE(timestamp), COUNT(*) FROM news GROUP BY DATE(timestamp);
```

### Q3: 看不到新闻数量徽章
**可能原因**：
- 确实没有新闻
- 日期格式不匹配

**解决方法**：
1. 确认数据库有新闻
2. 检查新闻的时间戳
3. 重新抓取新闻：`python news_scraper_v2.py`

### Q4: 月份切换后报错
**可能原因**：
- 那个月确实没有新闻
- 代码逻辑问题

**解决方法**：
1. 这是正常的（无新闻显示灰色）
2. 检查控制台错误信息
3. 查看详细日志
