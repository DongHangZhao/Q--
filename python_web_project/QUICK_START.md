## 🔧 常见问题

### Q1: 提示找不到模块
**错误**：`ModuleNotFoundError: No module named 'schedule'`

**解决**：
```bash
pip install schedule requests beautifulsoup4
```

### Q2: 抓取失败
**错误**：`抓取新华网失败：Connection timeout`

**解决**：
- 检查网络连接
- 确认能访问新华网（www.xinhuanet.com）
- 可能是临时网络问题，稍后重试

### Q3: 数据库为空
**现象**：新闻列表页显示"暂无新闻"

**解决**：
```bash
# 手动执行一次抓取
python news_scraper.py
```

### Q4: 定时任务未启动
**现象**：控制台没有显示"定时任务线程已启动"

**解决**：
- 检查 app.py 是否正确导入 scheduler
- 确认 init_scheduler() 被调用
- 重启应用
