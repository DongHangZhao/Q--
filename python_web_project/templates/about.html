平台介绍

咫尺天涯是一个集交友、新闻、视频、论坛于一体的综合性社交平台。我们致力于连接每一个有趣的灵魂，让世界变得更小，让交流变得更加便捷。

  • 实时社交互动
  • 新闻资讯获取
  • 视频分享娱乐
  • 兴趣社区交流
  • 个性化内容推荐

技术架构

  • 后端: Python Flask + SQLAlchemy
  • 前端: HTML5, CSS3, JavaScript
  • 样式: Bootstrap 5
  • 数据库: SQLite
  • 安全: Flask-Login, WTForms验证
  • 多媒体: Pillow图像处理
