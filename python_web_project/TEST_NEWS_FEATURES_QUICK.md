
```sql
SELECT id, title, source, timestamp 
FROM news 
WHERE DATE(timestamp) = '2026-03-15';
-- 应该显示那天的新闻
```

**测试查询 3：特定来源**
```sql
SELECT source, COUNT(*) as count 
FROM news 
GROUP BY source;
-- 应该显示各来源的新闻数量
```

**测试查询 4：包含关键字的新闻**
```sql
SELECT id, title, source 
FROM news 
WHERE title LIKE '%科技%' 
   OR content LIKE '%科技%';
-- 应该显示包含"科技"的新闻
```

## 🐛 常见问题排查

### Q1: 日历中没有蓝色日期
**原因**：数据库中没有新闻

**解决**：
```bash
python test_news_scraper.py
# 抓取新闻并保存
```

### Q2: 搜索不到任何结果
**可能原因**：
1. 数据库为空 → 运行抓取脚本
2. 关键字太特殊 → 换个常见的词
3. 日期选择错误 → 选择有新闻的日期

### Q3: 搜索很慢
**原因**：数据量大或没有索引

**解决**：
- 添加数据库索引
- 优化查询语句

### Q4: 点击日期报 404
**原因**：路由未注册

**解决**：
- 重启应用
- 确认 `routes/news.py` 已正确导入
