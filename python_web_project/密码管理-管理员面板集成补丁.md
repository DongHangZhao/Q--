### 1. 在 UserAdminView 类中添加密码管理功能

找到 `admin_panel.py` 文件中的 `UserAdminView` 类（约在295行），在其后添加以下方法：

```python
from flask import flash, redirect, request
from flask_admin import expose
from werkzeug.security import generate_password_hash
import random

class UserAdminView(AdminModelView):
    # ... 现有代码 ...
    
    def get_list_row_actions(self):
        """添加自定义操作按钮"""
        from flask_admin.model import template
        return [
            template.EditRowAction(),
            template.ViewRowAction(),
        ]
    
    @expose('/reset-password/')
    def reset_password(self):
        """重置用户密码"""
        from models import User, PasswordChangeLog
        
        user_id = request.args.get('id')
        user = User.query.get(user_id)
        
        if not user:
            flash('用户不存在', 'error')
            return redirect(self.get_url('.index_view'))
        
        # 生成8位随机密码
        new_password = ''.join([str(random.randint(0, 9)) for _ in range(8)])
        user.password_hash = generate_password_hash(new_password)
        
        # 记录日志
        log = PasswordChangeLog(
            user_id=user.id,
            changed_by='admin',
            ip_address=request.remote_addr,
            reason=f'管理员 {current_user.username} 重置密码'
        )
        db.session.add(log)
        db.session.commit()
        
        flash(f'已重置用户 {user.username} 的密码为: {new_password}，请通知用户登录', 'success')
        return redirect(self.get_url('.index_view'))
```

### 3. 添加密码字段的表单处理

```python
from wtforms import PasswordField

class UserAdminView(AdminModelView):
    # 在类定义中添加
    form_extra_fields = {
        'password': PasswordField('新密码（留空则不修改）')
    }
    
    def on_model_change(self, form, model, is_created):
        """保存时处理密码"""
        from werkzeug.security import generate_password_hash
        
        if hasattr(form, 'password') and form.password.data:
            model.password_hash = generate_password_hash(form.password.data)
```

```sql
-- 查看密码修改日志
SELECT l.*, u.username, l.timestamp, l.reason
FROM password_change_logs l
JOIN users u ON l.user_id = u.id
ORDER BY l.timestamp DESC
LIMIT 20;

-- 查看密码重置请求
SELECT r.*, u.username
FROM password_reset_requests r
JOIN users u ON r.user_id = u.id
ORDER BY r.created_at DESC
LIMIT 20;
```

## 注意事项

1. **密码加密**：管理员重置密码时会自动加密
2. **日志记录**：所有密码修改都会记录到数据库
3. **随机密码**：建议生成8位以上复杂密码
4. **通知用户**：重置密码后需要通知用户新密码
5. **首次登录**：建议要求用户首次登录后立即修改密码
