# 咫尺天涯 - 社交平台

咫尺天涯是一个功能丰富的社交平台，旨在为用户提供一个集新闻、视频、帖子分享、聊天于一体的综合社交体验。平台名称寓意着虽然人与人之间可能相隔万里，但通过网络可以实现心灵的贴近交流。

## 功能特点

### 1. 用户系统
- 用户注册和登录
- 个人信息管理（头像、简介、位置等）
- 用户间关注/取消关注功能

### 2. 内容系统
- **新闻功能**：用户可以浏览和点赞时事新闻
- **视频功能**：支持视频上传、播放和点赞
- **帖子功能**：用户可以发布图文帖子，支持点赞和评论

### 3. 互动功能
- 用户间私信聊天
- 点赞系统（对帖子、视频、新闻点赞）
- 关注其他用户

### 4. 管理功能
- 管理员账户系统
- 封禁功能（完全封禁、禁止发帖、禁止发视频、禁止评论）
- 用户权限管理

### 5. 热度排名
- 基于点赞数、评论数和发布时间的热度算法
- 今日热门内容展示

### 6. 响应式设计
- 完美适配桌面端和移动端
- 使用Bootstrap 5实现响应式布局
- 流畅的用户体验

## 技术栈

- **后端**: Python Flask框架
- **数据库**: SQLAlchemy ORM (SQLite)
- **前端**: HTML5, CSS3, JavaScript
- **UI框架**: Bootstrap 5
- **表单处理**: Flask-WTF
- **用户认证**: Flask-Login

## 安装与运行

1. 克隆项目到本地
2. 安装依赖：
   ```
   pip install -r requirements.txt
   ```
3. 运行应用：
   ```
   python run.py
   ```
4. 访问 `http://localhost:5000`

## 特色功能

1. **智能热度算法**：基于内容的点赞数、评论数和发布时间计算热度分数
2. **全面的权限管理**：管理员可精细化控制用户权限
3. **多媒体支持**：支持图片、视频等多种媒体格式
4. **实时互动**：用户间私信聊天功能
5. **响应式界面**：适配各种设备屏幕尺寸
