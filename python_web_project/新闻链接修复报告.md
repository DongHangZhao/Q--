## 🔧 技术实现

### 1. 新闻详情页按钮
**文件**：`templates/news_detail.html`

**代码**：
```html
{% if news.url %}
<div class="mt-4 pt-3 border-top">
  <a href="{{ news.url }}" target="_blank" rel="noopener noreferrer" 
     class="btn btn-primary btn-lg">
    <i class="fas fa-external-link-alt"></i> 查看原文（{{ news.source }}）
  </a>
  <small class="text-muted d-block mt-2">
    <i class="fas fa-info-circle"></i> 点击按钮将在新窗口打开原新闻网站
  </small>
</div>
{% endif %}
```

### 2. 新闻列表页按钮
**文件**：`templates/news.html`

**代码**：
```html
{% if news.url %}
<a href="{{ news.url }}" target="_blank" rel="noopener noreferrer" 
   class="btn btn-sm btn-outline-primary" onclick="event.stopPropagation();">
  <i class="fas fa-external-link-alt"></i> 查看原文
</a>
{% endif %}
```

### 3. 安全措施
- `target="_blank"`：在新窗口打开
- `rel="noopener noreferrer"`：防止钓鱼攻击
- `onclick="event.stopPropagation();"`：避免事件冲突
