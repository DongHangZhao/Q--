# 密码管理系统 - 快速使用指南

## 🎯 立即体验

### 1. 修改密码
```
访问地址：http://127.0.0.1:5000/password/change-password
前提条件：需要登录
```

### 2. 忘记密码
```
访问地址：http://127.0.0.1:5000/password/forgot-password
前提条件：无需登录
特点：带图像验证码验证
```

### 3. 验证重置码
```
访问地址：http://127.0.0.1:5000/password/verify-code
前提条件：已从管理员处获得6位重置码
```

### 4. 重置密码
```
访问地址：http://127.0.0.1:5000/password/reset-password?user_id=1
前提条件：已成功验证重置码
```

## 📖 完整流程示例

### 用户忘记密码的完整流程

**第1步：提交申请**
1. 打开 http://127.0.0.1:5000/password/forgot-password
2. 输入用户名：`testuser`
3. 输入图像验证码：`ABCD`（看图片输入）
4. 点击"提交重置请求"
5. 系统显示：**您的重置码为: 123456**
6. 复制并保存这个重置码

**第2步：联系管理员**
通过以下方式联系管理员：
- 站内消息："您好，我忘记密码了，我的重置码是123456"
- 或者邮件、电话、微信等

**第3步：管理员验证**
管理员操作：
1. 登录管理员账号
2. 访问用户管理页面
3. 找到用户 `testuser`
4. 点击"查看重置码"
5. 确认用户提供的重置码一致

**第4步：验证重置码**
1. 打开 http://127.0.0.1:5000/password/verify-code
2. 输入用户名：`testuser`
3. 输入重置码：`123456`
4. 点击"验证"
5. 验证成功，自动跳转

**第5步：设置新密码**
1. 输入新密码：`newpass123`
2. 确认新密码：`newpass123`
3. 点击"重置密码"
4. 成功！自动跳转到登录页

**第6步：登录**
使用新密码 `newpass123` 登录系统

## 🔧 管理员操作

### 查看用户的重置码
```javascript
// 使用浏览器开发者工具或Postman
fetch('/password/admin/view-reset-code', {
  method: 'POST',
  headers: {'Content-Type': 'application/x-www-form-urlencoded'},
  body: 'user_id=1'
})
.then(res => res.json())
.then(data => console.log(data));

// 返回示例：
// {
//   "success": true,
//   "message": "用户 testuser 的重置码: 123456",
//   "verification_code": "123456",
//   "expires_at": "2026-04-29 16:30:00"
// }
```

### 直接重置用户密码
```javascript
fetch('/password/admin/reset-user-password', {
  method: 'POST',
  headers: {'Content-Type': 'application/x-www-form-urlencoded'},
  body: 'user_id=1&new_password=temp123456'
})
.then(res => res.json())
.then(data => console.log(data));

// 返回示例：
// {
//   "success": true,
//   "message": "已重置用户 testuser 的密码"
// }
```

## 📊 数据库查询

### 查看所有重置请求
```sql
SELECT * FROM password_reset_requests 
ORDER BY created_at DESC 
LIMIT 10;
```

### 查看密码修改日志
```sql
SELECT l.*, u.username 
FROM password_change_logs l
JOIN users u ON l.user_id = u.id
ORDER BY l.timestamp DESC 
LIMIT 10;
```

## 🛡️ 安全说明

### 3层验证机制
1. **图像验证码**：防止机器人攻击
2. **6位重置码**：30分钟有效，一次性使用
3. **管理员验证**：人工确认用户身份

### 密码要求
- 最少6位
- 建议包含字母和数字
- 不使用常见密码

### 日志记录
所有密码修改操作都会记录：
- 修改者（用户/管理员/系统）
- IP地址
- 时间戳
- 修改原因
