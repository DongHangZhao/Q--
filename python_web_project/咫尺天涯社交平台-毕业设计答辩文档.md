### 3.2 数据库连接代码

#### models/__init__.py - 数据库初始化
```python
from flask_sqlalchemy import SQLAlchemy
from datetime import datetime, timezone

# 创建SQLAlchemy实例
db = SQLAlchemy()

class FlexibleDateTime(db.TypeDecorator):
    """自定义日期时间类型，支持时区处理"""
    impl = db.DateTime
    
    def process_bind_param(self, value, dialect):
        if value is not None:
            if value.tzinfo is None:
                # 添加本地时区
                return value.replace(tzinfo=timezone.utc)
            return value.astimezone(timezone.utc)
        return value
    
    def process_result_value(self, value, dialect):
        if value is not None:
            return value.replace(tzinfo=timezone.utc)
        return value
```

#### app.py - 数据库配置
```python
# 数据库配置
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///./instance/app.db'
app.config['SQLALCHEMY_TRACK_MODIFICATIONS'] = False

# 初始化数据库

# 创建数据库表（首次运行）

    db.create_all()
```

### 4.2 创建操作（INSERT）

#### 创建新用户
```python
from werkzeug.security import generate_password_hash

new_user = User(
    username='lisi',

```

User.query.filter(User.join_date < '2025-01-01').update({User.is_active: False})

#### 修改帖子内容
```python
post = Post.query.get(1)
post.title = '更新后的标题'
post.content = '更新后的内容'

db.session.commit()
```

### 4.4 删除操作（DELETE）

#### 删除单个记录
```python
# 删除用户
user = User.query.get(1)
db.session.delete(user)

# 删除帖子
post = Post.query.get(1)
db.session.delete(post)
db.session.commit()
```

#### 批量删除
```python
# 删除所有未激活用户
inactive_users = User.query.filter(User.is_active == False).all()
for user in inactive_users:
    db.session.delete(user)
db.session.commit()
```

### 6.1 细粒度权限管理

#### 四种权限控制方式
| 权限类型 | 控制字段 | 状态值 | 说明 |
|----------|----------|--------|------|
| **完全封禁** | `is_banned` | True/False | 无法登录系统 |
| **发帖权限** | `can_post` | True/False | 是否可以发布动态 |
| **发视频权限** | `can_video` | True/False | 是否可以上传视频 |
| **评论权限** | `can_comment` | True/False | 是否可以发表评论 |

#### 权限管理流程图
```
用户点击权限管理
        ↓
显示当前权限状态
        ↓
根据状态显示不同选项
        ↓
点击"禁止发帖" → can_post=False
点击"允许发帖" → can_post=True
点击"全部解封" → 所有权限恢复
```

### 8.1 必须掌握的问题

#### 数据库相关
- Q：数据库是如何连接的？
- A：通过SQLAlchemy配置URI，使用`db.init_app(app)`初始化

- Q：如何查询用户的所有帖子？
- A：`Post.query.filter_by(author_id=user.id).all()`

- Q：如何实现软删除？
- A：添加`is_deleted`字段，查询时过滤`is_deleted==False`

#### 权限管理相关
- Q：四种权限是如何实现的？
- A：在User模型中添加四个布尔字段，通过后台路由控制

- Q：为什么需要单独的权限管理？
- A：比完全封禁更灵活，满足不同管理需求

#### 安全相关
- Q：密码是如何存储的？
- A：使用PBKDF2-SHA256哈希算法，不存储明文密码

- Q：如何防止CSRF攻击？
- A：使用Flask-WTF的CSRFProtect，表单中包含csrf_token
