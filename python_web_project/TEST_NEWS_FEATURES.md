```sql
-- 查看总新闻数
SELECT COUNT(*) FROM news;

-- 查看最新 10 条新闻
SELECT id, title, source, timestamp 
FROM news 
ORDER BY timestamp DESC 
LIMIT 10;

-- 查看某天的新闻
SELECT * FROM news 
WHERE DATE(timestamp) = DATE('now');

-- 按月统计
SELECT 
    DATE(timestamp) as date,
    COUNT(*) as count
FROM news
GROUP BY DATE(timestamp)
ORDER BY date DESC;
```

## 📋 常见问题排查

### Q1: 日历视图仍然报错
**解决**：
1. 确认 `routes/news.py` 已修改
2. 重启应用
3. 清除浏览器缓存

### Q2: 点击日期 404
**解决**：
1. 确认 `news_by_date` 路由已添加
2. 检查 URL 格式是否正确
3. 查看控制台错误信息

### Q3: 新闻详情页链接错误
**解决**：
1. 确认 `news_detail.html` 已修改
2. URL 应该是 `news_routes.news_by_date`
3. 重启应用
