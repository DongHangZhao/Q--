## ⚠️ 需要注意的问题

### 数据库管理后台视图（已禁用）
**原因**: `FlexibleDateTime`自定义类型与Flask-Admin过滤器不兼容
**影响**: 无法在dbadmin中直接查看密码重置请求和日志表
**解决方案**: 
1. 使用管理员面板查看密码日志
2. 或直接查询数据库
3. 或使用SQLite浏览器查看

**管理员仍然可以**:
- ✅ 在管理员面板重置用户密码
- ✅ 在管理员面板查看密码日志
- ✅ 通过API查看所有重置请求

## 🎯 使用指南

### 用户忘记密码流程

**步骤1**: 访问登录页
```
http://127.0.0.1:5000/login
```
👉 点击"忘记密码？"链接

**步骤2**: 提交重置请求
```
http://127.0.0.1:5000/password/forgot-password
```
- 输入用户名
- 完成图像验证码
- 获取6位重置码（例如：123456）

**步骤3**: 联系管理员
- 提供重置码给管理员
- 管理员验证身份

**步骤4**: 验证重置码
```
http://127.0.0.1:5000/password/verify-code
```
- 输入用户名
- 输入6位重置码
- 验证成功

**步骤5**: 设置新密码
```
http://127.0.0.1:5000/password/reset-password
```
- 自动跳转（验证成功后）
- 输入新密码
- 确认新密码
- 完成

---

### 用户修改密码

**方式1**: 通过编辑资料页
```
http://127.0.0.1:5000/edit_profile
```
👉 点击蓝色提示框中的"修改密码"链接

**方式2**: 直接访问
```
http://127.0.0.1:5000/password/change-password
```

### 管理员重置密码

**步骤1**: 访问管理员面板
```
http://127.0.0.1:5000/admin/users
```

**步骤2**: 找到目标用户

**步骤3**: 点击"重置密码"按钮
- 系统生成8位随机密码
- 显示新密码
- 通知用户

**步骤4**: （可选）点击"密码日志"查看历史记录
